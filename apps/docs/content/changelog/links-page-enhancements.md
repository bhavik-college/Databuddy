- Replaced link dialog with full-height sheet panel for create/edit
- Added QR code tab in edit sheet with customizable options
- QR codes support resolution selection (128px, 256px, 512px, 1024px)
- QR codes support style selection (squares or dots with rounded corners)
- QR codes support 8 color presets for foreground color
- QR codes support logo upload with adjustable size slider
- Added `react-qrcode-logo` package for advanced QR customization
- Added search bar with debounced filtering by name, slug, or target URL
- Added sort options (newest, oldest, name A-Z, name Z-A)
- Added QR code action in link item dropdown menu
- Added standalone QR code dialog for quick access
- Added `prefix` and `suffix` props to Input component for clean addon support
- Focus indicator on prefixed inputs now only spans the input field
